DataScienceAPIDocumentation

This repo houses all published Data Science API documentation.

You will need OAuth core credentials to use these services. (If you do not have these please contact API and Auth to request credentials)

We provide .net and java wrappers for our APIs, which can be found here.

We monitor our services and publish a very simple page with our current status with documentation here.

Our general API versioning documentation can be found here.

APIs:

Geography: Provides geolocation data.

JobTitle: Provides normalized job titles like Carotene and oNet.

Skills: Provides normalized skills.

ResumeParsing: Parses resumes and returns contained data through several parsers.

ParseAndNormalize: Parses resumes and normalizes through most other services offered.

JobLevel: Provides a normalized job level for a job title.

KeywordStuffingDetector: Indicates if a job has been keyword stuffed.

RelevantSkills: Provides a list of relevant skills for a given skill.

ResolvedTitle: Attempts to resolve a job title to the closest match from a given list of target titles.
